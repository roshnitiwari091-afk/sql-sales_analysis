# sql-sales_analysis
Retail Sales Data Analysis using PostgreSQL
Dataset: 9,994 retail transactions

##Analysis performed:
• Annual sales trend
• Sales growth analysis
• Monthly sales aggregation
• Regional contribution
• Category profitability

##Tools used:
PostgreSQL
SQL Window Functions
## Key Insights
• Sales grew 51% between 2014 and 2017
• West region contributes 31.6% of total revenue
• Technology category has highest profit margin (17.4%)
